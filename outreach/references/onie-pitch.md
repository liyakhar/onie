# Onie — canonical pitch (for outreach)

Use these lines when mentioning Onie in replies or DMs. Pick one fit; don't stack them.

## One-liner

**Onie** is a public feed where practitioners publish the agent workflows they actually run — prompts, skills, file layouts, and setups, tagged by discipline and stack.

## URL

Production: https://onie-web-production.up.railway.app  
Short: onie.app (when live)

## Why someone would cross-post

- Their thread gets buried; a feed post stays discoverable
- Others in their field can search by tool and discipline
- No paywall — publish and link back to their repo or gist

## What Onie is not

- Not a prompt marketplace or course
- Not claiming ownership of their workflow
- Not asking them to delete the X thread — invite is optional

## Fields we name explicitly

UX, SaaS, science, engineering, and other disciplines where people run agents in real work.
